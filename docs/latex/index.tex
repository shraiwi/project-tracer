At its current stage, it is designed for the E\+S\+P32, a low-\/power microcontroller with Wi\+Fi and Bluetooth. However, the code is written to be portable and can easily be ported to almost any platform.\hypertarget{index_autotoc_md0}{}\doxysubsection{Features}\label{index_autotoc_md0}
