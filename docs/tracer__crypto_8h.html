Performs a AES-CTR operation on a variable-length block.

Defined as AES128-CTR(key, iv, data) in the standard. This operation is symmetrical, like XORing. If the operation is performed twice on the same data (with the same arguments) the original data can be recovered.

Parameters
keyA pointer to a buffer containing the key.
key_lenThe size of the key buffer, in bytes.
ivA pointer to a 16-byte buffer containing the initialization variable
dataA pointer to a buffer containing the data to flip.
data_lenThe size of the data buffer, in bytes.
outputA pointer to where the output buffer is located. If NULL, the function will allocate a buffer of size data_len and write the decrypted data there.
Returns
A pointer to a buffer containing the flipped data.
